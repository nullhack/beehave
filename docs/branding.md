# Branding — beehave

## Identity

- **Project name:** beehave
- **Tagline:** BDD living documentation in sync
- **Tone of voice:** Friendly, precise, industrious, collaborative

## Visual

- **Primary color:** `#1e2128` — deep charcoal with cool undertones (dark glass, professionalism)
- **Accent color:** `#f5a800` — warm amber/gold (bee energy, optimism, pytest-beehave heritage)

> Colors meet WCAG 2.1 AA (≥ 16:1 contrast) when white text is placed on the primary.

## Release Naming

- **Convention:** Adjective-animal (e.g., "Busy Badger", "Diligent Dolphin")
